#    ----------------->  Strings <------------------
<br>
* Strings : A character array terminated by a '\0' ( null character )
-           null character denotes string termination.

-> For example : 
->             char name[] = { 'R','A','M','A','N','A','N','D','\0'};
->               OR
->             char name[] = "RAMANAND";
